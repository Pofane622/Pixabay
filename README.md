Pixabay Clone

A Pixabay-inspired image gallery built with React. Search, browse, and download royalty-free images using the Pixabay API.

Live Demo: https://reliable-blancmange-55faf2.netlify.app/

Features

- Search for royalty-free images
- Browse categories like nature, forest, sky, beach
- View image details including likes, views, downloads
- Responsive grid layout
- Dark and light mode toggle
- Hover effects on images

React, React Router, Context API, CSS, Pixabay API

git clone https://github.com/Pofane622/Pixabay.git
cd pixabay-clone
npm install
npm start
